# BugPet Tauri 重构日志
启动:
cd e:\project\BugPet\BugPet\bugpet
npm run tauri dev


## 项目背景

**原始项目**：BugPet - 一个 macOS 桌面宠物应用，使用 Swift + AppKit + SpriteKit 开发
**重构目标**：将 BugPet 从 macOS 原生应用迁移到 Windows 平台，使用 Tauri 框架实现跨平台支持

## 技术栈选择

| 类别 | 技术 | 版本 | 选型理由 |
|------|------|------|----------|
| **前端** | Vue 3 | ^3.5.13 | 响应式 UI，组件化开发 |
| **前端** | TypeScript | ~5.6.2 | 类型安全，代码质量 |
| **前端** | Tailwind CSS | ^4.2.2 | 快速样式开发，响应式设计 |
| **后端** | Rust | 1.94.1 | 高性能，跨平台，系统 API 调用 |
| **框架** | Tauri | ^2 | 轻量级，低资源占用，原生窗口 |
| **构建** | Vite | ^6.0.3 | 快速开发，热更新 |
